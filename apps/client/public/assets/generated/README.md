# Generated Asset Drop Zone

Generated raster images, textures, sprites, and exported model previews belong here when they are not authored by hand.

Replacement rules:
- Keep filenames stable once referenced by `/assets/manifest.json`.
- Prefer PNG or WebP for sprites/textures and GLB for tank models.
- Include source prompts, model/tool names, and license notes in this README or `THIRD_PARTY_NOTICES.md`.
- Gameplay must keep working if any generated asset is missing; the renderer falls back to procedural meshes/materials.

Current status: no generated images or models are required for the playable slice.

Audio note: `../audio/ambient-music.wav` is a user-provided local project asset and is referenced from `/assets/manifest.json`.
